Movie Database


Documentation

Quick Start

The purpose of this API is for users to quickly find their favorite actors, directors and movies.
This can help the individual find more movies those actors were in, other films directed by their favorite director,
or other movies in the selected genre.

Users can create profiles to save information about their favorite movies so it is easily accessible

Resource URL

https://github.com/scott-a-hall/movie_api
